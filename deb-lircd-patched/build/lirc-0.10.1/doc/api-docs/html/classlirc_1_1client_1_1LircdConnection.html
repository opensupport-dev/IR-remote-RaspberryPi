Detailed Description

Interface to receive lircrc-translated keypresses.

This is basically built on top of lirc_code2char() and as such supporting centralized translations using lircrc_class. See lircrcd(8).

Parameters
  • program: string, used to identify client. See ircat(1)
  • lircrc: lircrc file path. See get_default_lircrc_path() for defaults.
  • socket_path: lircd output socket path, see get_default_socket_path() for defaults.
